The following is the ReadMe file of 'Tic Tac Toe'
================================================================
1. System Requirements
2. How to install 
================================================================
1. System Requirements:
- Any version of python which supports pygame
- Pygame library

2. How to install:
- To download latest python version go to https://www.python.org
- Install pygame using command 'pip install pygame' in terminal/command prompt
- In case .exe fails to install all dependencies use the zip file to manually extract them
